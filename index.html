How To Use

1

Run the Python backend locally or on a server: python .\app.py.

2

Enter the backend URL here. Local default is http://127.0.0.1:8080.

3

Use the backend dashboard to edit configuration, start/stop the bot, and view applied/skipped jobs.

Why It Is Not Running On Netlify

Netlify cannot run this bot by itself because the bot is not a static website. It needs a long-running Python process, local CSV files, Selenium, and a real Chrome browser session.

  • http://127.0.0.1:8080 means “this computer.” It works only if you run python .\app.py on the same computer where you opened the browser.
  • For other users to access it, deploy the Python backend to a VPS/server that supports Chrome and use that public HTTPS backend URL here.
  • Use Netlify for the frontend only; use a server/VPS/local machine for the bot backend.
